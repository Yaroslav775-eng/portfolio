Мета: Масштабувати топові категорії у пік сезону, захищаючи маржу.

Стратегія

Категорійне зростання. Впроваджено схему категорійного зростання через багатоканальну воронку. Замінено узагальнені структури чіткими товарно-категорійними шарами попиту (наприклад, вугільні грилі vs аксесуари) для максимізації ROI на високомаржинальних товарах.

Прийняті рішення

  • Категорійна декомпозиція: Сегментація інвентарю за рівнями маржі для запобігання неприбутковому масштабуванню та фокусу бюджету на затребуваних категоріях.
  • Масштабування через Performance Max: Використання PMax як головного драйвера всередині категорій для автоматизованого охоплення суміжної аудиторії.
  • Гібридний Search-шар: Збереження пошукових кампаній як прецизійного інструменту для "гарячого" попиту, що доповнює широку логіку PMax.
  • Структурна реорганізація: Побудова прозорої товарної логіки всередині акаунту для виключення внутрішньої конкуренції між групами товарів.

Стратегія

Прибутковість через інвентар. Впроваджено систему пріоритетів, де рекламні інвестиції слідують за прибутком бізнесу, що стабілізувало управління performance-каналами для величезного каталогу SKU.

Прийняті рішення

  • Інвентарна пріоритезація: Виділення ключових товарних груп в окремі "ударні" кампанії з пріоритетним доступом до бюджету на основі маржі.

  • Ешелонований захист бренду: Ізоляція брендового пошуку в окремий шар для запобігання "роздуванню" звітів PMax органічним трафіком.
  • Гранулярний перехід: Ліквідація "загального потоку" по всьому каталогу шляхом впровадження прозорої товарної логіки за брендами та категоріями.
